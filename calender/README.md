## Installation

1. Install dependencies:
```bash
pip install -r requirements.txt
```

2. Set up your Google Gemini API key:
   - Create a `.env` file in the project directory
   - Add your API key: `GOOGLE_API_KEY=your_api_key_here`

3. Run the application:
```bash
streamlit run app.py
```

## Usage

### 1. Select Language
- Use the sidebar to choose your preferred language (English/Hindi/Marathi)
- **Note**: Plans generated by AI are automatically translated to your selected language
- You can change language anytime - all existing plans will be translated

### 2. Create a Farming Plan
- Enter your farming question or task
- Click "Generate Plan" to get AI-powered recommendations
- Review and edit the plan steps as needed

### 3. Schedule Events
- Select a date and time for your farming task
- Click "Add to Calendar" to save the event

### 4. Manage Events
- Navigate through months using Previous/Next buttons
- Click on any event to view details
- Edit or delete events as needed
